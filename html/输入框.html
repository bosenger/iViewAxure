Input 输入框

概述

基本表单组件，支持 input 和 textarea，并在原生控件基础上进行了功能扩展，可以组合使用。

基础用法

由于Axure对于输入框样式的支持很弱，所以输入框使用形状和输入框联合实现。

尺寸

输入框有三种尺寸：大、默认（中）、小

可清空

通过输入框的Onfocus事件制作的清空效果



带Icon的输入框

可以在背景框中输入字符或图标来增加指示内容



前缀和后缀图标

通过设置背景边框左或右对齐并调整输入框位置来实现





搜索框

在输入框的右边增加按钮，并调整圆角即可实现





文本域

Axure无法支持支持拖拽调整文本域大小的同时改变边框，所以调整大小的动效暂时不支持。

禁用状态

由于文本框的disabled状态和外框的disabled状态无法同步样式，所以对于禁用状态的输入框单独做了一份样式。
