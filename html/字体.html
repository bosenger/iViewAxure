Font 字体

概述

iView 的 CSS 对字体进行了统一规范，力求在不同平台、浏览器下能显示出其最佳的效果。我们推荐  macOS（iOS）优先的策略，在不支持苹方字体的情况，使用备用字体。

字体预览

• 中文字体

用数据说话

PingFang SC

MAC OS 优先

Axure中优先选用了与MAC OS更兼容的苹方字体，在Windows中可能会显示不出该字体的效果，建议单独安装苹方字体

• 英文字体

iView UI

PingFang SC

MAC OS 优先

由于Axure没有font-family的机制，所以英文字体任然沿用苹方字体。
